## 安装使用

### 拉取代码

#### 从 `GitHub` 上拉取

```bash
git clone https://github.com/atorber/x-admin.git
```

### 安装依赖

```bash
cd x-admin

pnpm install
```

### 启动平台

```bash
pnpm dev
```

### 项目打包

```bash
pnpm build
```
